# Privacy Policy

Last updated: 2026-04-27

## Overview
Haro is a Chrome extension that Turn your browser into an AI browser.

## Data Collection
This extension does NOT collect, store, or transmit any personal data 
to external servers.

## Data Usage
- All data processed by this extension remains locally on your device.
- No data is shared with third parties.
- No analytics or tracking tools are used.

## Permissions
This extension requires certain browser permissions solely to provide 
its core functionality:
- **activeTab**: To access the current tab when user activates the extension.
- **debugger**: To [你的用途].
- **scripting**: To inject scripts for [你的用途].
- **sidePanel**: To display the extension UI in the side panel.
- **storage**: To save user preferences locally.
